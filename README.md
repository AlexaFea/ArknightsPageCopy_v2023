## 页面布局分析
采用t字形布局。主要内容共占5次滚动，分别展示：游戏logo与主角，游戏最新公告，角色介绍，世界观介绍和其他内容。将账号登录、注册和分享放置在页首，与导航栏同样使用固定定位。当主要内容到达尾部时再进行滚动时页首和导航栏上滚，展示页脚。
